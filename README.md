6. IoC / DI 에 대해 간략하게 설명해 주세요!

    IoC는 제어의 역전(프로그램의 제어 흐름이 뒤바뀜)

    DI는 의존성 주입, IoC의 한 형태로 객체를 직접 생성하는 것이 아니라 의존 객체를 주입 받는다.
